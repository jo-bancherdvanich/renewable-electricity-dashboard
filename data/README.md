# Data

Raw national datasets used in the project. China's OWID source file is very large
(~14 MB) and is not included here; download it from
https://ourworldindata.org/profile/energy/china

- `Australia.xlsx` — DCCEEW Australian Energy Statistics, Table O
- `NewZealand.xlsx` — MBIE Electricity Statistics
- `Germany.pdf` — Umweltbundesamt / AGEE-Stat time series (values extracted manually)
- `cleaned_model_data.xlsx` — the cleaned, combined dataset loaded into the Power BI model

See ../docs/data-decisions-log.md for how each source was cleaned and standardised.
